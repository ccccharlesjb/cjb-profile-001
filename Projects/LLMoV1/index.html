LLMo Version 1 Software

An Android mobile application designed to control the LLMO V1 robotic therapy bed, enabling automated physical therapy for the trunk and lower-limb extremities.

Unity C# MySQL

Features

Automated Exercises

Automates physical therapy exercises using the LLMo V1 robotic system. Users simply create a customized list of exercises tailored to each patient’s therapy session, and the system executes them automatically.

Patient Data Recording

Therapy sessions are recorded within the application, allowing therapists to manage patient profiles, add session notes, and configure individualized range-of-motion limits for safe treatment.

Wireless Connectivity

Provides wireless control of the LLMo V1 via Wi-Fi, enabling convenient operation without the need for physical connections.
