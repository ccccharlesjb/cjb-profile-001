Document Generator Tool

A desktop application for generating and organizing administrative documents through configurable templates, designed to reduce errors and streamline business workflows.

.NET C#

Features

Simplified Generation

Provides a guided workflow where users select the process they need to perform, choose the required documents, and complete structured input fields, significantly reducing complexity and preparation time.

Templated Documents

Supports fully customizable document templates based on editable .docx files. Users can modify templates externally and re-upload them into the system, enabling flexibility without requiring software changes.
