# SvelteKit Portfolio (Astrofy Port)

A high-performance, responsive portfolio and CV website built with **SvelteKit** and **Tailwind CSS**.

### [View Live Website](https://ccccharlesjb.github.io/cjb-profile-001/)

## The Project
This is a personal **reverse-engineering** project. I took the design and structure of the [Astrofy template](https://github.com/manuelernestog/astrofy) by Manuel Ernesto Garcia and manually ported it to **SvelteKit**. 

The goal was to practice Svelte's reactivity and component-based logic by rebuilding a professional-grade layout from scratch.

## Tech Stack
* **Framework:** [SvelteKit](https://kit.svelte.dev/)
* **Styling:** [Tailwind CSS](https://tailwindcss.com/)
* **Adapter:** `@sveltejs/adapter-static` (Optimized for GitHub Pages)

## Setup & Development

To get this project running on your local machine, follow these steps:

### 1. Installation
Install all necessary dependencies using npm:
```bash
npm install
```

### 2. Development Mode
Start the local development server to see changes in real-time.
```bash
npm run dev
```

### 3. Build & Preview
To generate the static files for production (GitHub Pages) and preview the final build locally:
```bash
# Build the project
npm run build

# Preview the built site
npm run preview
```
